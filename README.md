# Simple Music Player

一款界面部分仿照网易云的音乐播放器。


## 实现的功能

[x] 本地音乐扫描、播放（常见的音乐格式，使用旧版的 MediaPlayer 实现）
[x] 在线音乐获取、播放（使用网易云的音乐 API ，仅支持免费音乐）
[x] 通知栏控制的实现
[x] MediaSession 的接入，可以使用系统 API 控制音乐的播放、暂停和上下曲切换
